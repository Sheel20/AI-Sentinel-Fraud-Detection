🛡️ AI Sentinel
Google Solution Challenge 2026

📝 Project Overview
AI Sentinel is an Agentic RAG system designed to detect banking fraud in real-time. By utilizing specialized AI agents, it analyzes transaction behavior and security red flags to protect users, aligning with UN SDG 16.

🛠️ Tech Stack
Framework: LangGraph (Multi-agent orchestration)

Intelligence: Llama 3.2 via Ollama / Gemini 1.5 Flash

UI: Streamlit

Language: Python

🚀 How to Run Locally
Clone the repo: git clone https://github.com/Sheel20/AI-Sentinel-Fraud-Detection.git

Install requirements: pip install -r requirements.txt

Run the app: streamlit run app.py
